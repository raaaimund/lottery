# lottery

This is a Next.js app where you have to guess 4 numbers.

The following technologies were used:

- Next.js
- React.js
  - Context API
  - useState
  - useRef
- TypeScript
- tailwindcss

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
